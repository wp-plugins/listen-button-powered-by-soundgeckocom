Allows you to automatically display SoundGecko 'Listen' buttons next to your blog posts.

== Description ==

This plugin allows you to add the [SoundGecko](http://soundgecko.com/) 'Listen' button to each post of your WordPress blog that when clicked allows readers to hear your post read out.

This plugin benefits your site by expanding its distribution to a broader audience and making it more accessible to readers.

You can see an example on the [i started something blog](http://istartedsomething.com/).

You can automatically insert the buttons below or above your blog posts or you can use the template tag to insert the links wherever you like.

You can change the size, text, placement and whether it appears on pages in addition to posts.

To use the template tag, add the tag inside the WordPress loop: <?php do_action('soundgecko_listen_button'); ?>

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `soundgeckolisten.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure the plugin from the `Settings > SoundGecko listen button` administration page
